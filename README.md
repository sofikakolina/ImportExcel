# Excel Import App

Приложение для импорта, просмотра, редактирования и экспорта организационных данных из Excel.

## Запуск

1. Убедитесь, что установлены Docker и Docker Compose.
2. Скопируйте `.env.example` в `.env` и при необходимости измените переменные.
3. Выполните в корне проекта:
   ```bash
   docker-compose up --build
